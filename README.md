# 🕒 Digital Clock Using Python

A simple and interactive Digital Clock application built with Python and Tkinter that displays the current time in real-time through a graphical user interface (GUI).

## 🚀 Project Overview

This project demonstrates how to create a live digital clock using Python's built-in Tkinter library for GUI development and the Time module for real-time updates.

The clock automatically refreshes every second, providing an accurate and visually appealing display of the current system time.

This project is ideal for beginners who want to learn the fundamentals of GUI programming, event-driven applications, and real-time updates in Python.

## ⚙️ How It Works

The application:

1. Retrieves the current system time using Python's time module.
2. Displays the time inside a Tkinter window.
3. Updates the displayed time every second using Tkinter's scheduling functionality.
4. Continuously refreshes without requiring user interaction.
